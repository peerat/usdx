# Changelog

Все заметные изменения этого форка фиксируются здесь. Формат простой: свежие
изменения сверху, без привязки к релизным тегам, пока теги не заведены.

## 2026-05-13

### Добавлено

- Основной файл прошивки переименован в `R3VAF_uSDR_7_01v.ino`, чтобы имя
  соответствовало актуальной локальной версии сборки.

### Изменено

- `QUAD` вынесен в обычную опцию конфигурации и по умолчанию отключен
  комментарием в шапке файла, как и другие спорные/редко используемые функции.
- При выключенном `QUAD` его пункт больше не попадает в меню, а индикатор не
  занимает место на строке статуса.
- README и русская пользовательская документация синхронизированы с текущим
  состоянием прошивки `7.01v`: обновлены имя файла, экран, меню, дефолты и
  размеры памяти.

## 2026-05-11

### Добавлено

- Добавлен пункт `RX -> ScanStop` для остановки автопрокрутки по уровню сигнала.
  Значение `0` выключает остановку, `1..9` задает порог.
- Пользовательское руководство расширено до полноценного описания экрана,
  кнопок, меню, автопрокрутки, ScanStop, VOX, CW, CAT и памяти.

### Изменено

- Версия прошивки поднята до `4.00e`, чтобы новый EEPROM-параметр был
  корректно пересохранен.
- S-meter уровень теперь обновляется даже при выключенном отображении S-meter,
  чтобы `ScanStop` мог работать без отдельного индикатора на экране.
- `ScanStop` переведен на адаптивную остановку: прошивка запоминает минимальный
  и максимальный уровни за проход и останавливается на подъеме над минимумом.
- В меню `ScanStop` теперь отображается как `ScanStp` со значением относительного
  порога, например `+6dB`, а не как условный индекс.
- Повторный вход в меню теперь возвращает на последний выбранный раздел и пункт.
- Пауза анализа уровня при автоскане уменьшена и дополнена пропуском первых
  окон после перестройки, чтобы анализировать уже новый уровень частоты.
- VOX по умолчанию выключен и не запускает передачу во время работы с меню.
- Кнопка валкодера из обычного режима больше не открывает меню; меню открывает
  левая кнопка.
- Пункт `Keyer Mode` переименован в `Keyer`.
- CW-диагностический вывод отключен, чтобы не перекрывать экран в CW.
- Документация по памяти обновлена под текущий размер прошивки.
- Удален старый активный экран выбора групп меню; меню сразу работает с
  последним выбранным разделом.
- Повторяющаяся логика переключения режима вынесена в общие функции.
- S-meter разделен на сбор пика, расчет dBm и вывод на LCD; чтение уровня из
  ISR теперь атомарное.
- `ScanStop` перед проверкой порога принудительно пересчитывает свежий dBm
  после паузы анализа и пропускает первые окна уровня после шага, чтобы не
  остановиться на старом пике от предыдущей частоты.
- Исправлена опечатка `smeter > 0` на `swrmeter > 0` в CW/SWR ветке.
- Исправлен вход и выход из меню: левая кнопка гарантированно открывает меню,
  кнопка валкодера из обычного экрана меню не вызывает, а после выхода
  принудительно перерисовываются баннер, VFO и курсор шага.
- Переход между разделами меню теперь выполняется прокруткой через край
  текущего раздела, без отдельного экрана выбора групп.
- При редактировании `Mode` через меню теперь сохраняются и режим VFO, и
  связанный фильтр/шаг.
- Во время автопрокрутки экранный `dBm` обновляется быстрее; проверка
  `ScanStop` по-прежнему ждет стабилизацию после шага.
- `ScanStop` больше не останавливает скан навсегда: всплеск сначала
  подтверждается, затем частота удерживается около 10 секунд, после чего скан
  продолжается без действий пользователя.
- Время оценки уровня после шага уменьшено до `120 ms`; подтверждение сигнала
  занимает около `350 ms`.
- Во время автопрокрутки курсор шага частоты скрывается, чтобы убрать
  мельтешение в нижней строке рядом со статусом `R`.

### Проверено

- `platformio run -e uno`
- `tools/size_profiles.sh`
- RAM: `1430 / 2048 B`
- Flash: `32208 / 32256 B`

## 2026-05-10

### Добавлено

- Указано, что проект является форком uSDX/uSDR, а не upstream-прошивкой.
- Добавлена русская пользовательская документация `docs/USER_GUIDE_RU.md`.
- Добавлен этот журнал изменений.
- Меню сгруппировано в английские разделы `General`, `RX`, `TX`, `CW`.
- Вход в подменю и редактирование сделаны через кнопку валкодера.
- Добавлена инерционная автопрокрутка частоты по темпу ручного вращения
  валкодера.
- При активной автопрокрутке любое действие валкодером только останавливает
  сканирование.

### Изменено

- Обновлены дефолтные настройки под текущий сценарий работы:
  VFO A `7074 kHz LSB`, VFO B `14074 kHz USB`, громкость `9`, CAT `ON`,
  TX Drive `8`, PA max `220`, CW decoder `ON`, keyer `20 WPM`, `IambicA`.
- CW-декодер доработан для более терпимой работы с нестандартными паузами между
  элементами, буквами и словами.
- В CW-режиме при включенном декодере на экране показывается строка
  декодированного текста.
- Документация по сборке обновлена под текущий размер прошивки.
- Навигация по подменю переведена на компактные таблицы пунктов в flash.
- Повторное сохранение VFO mode/filter вынесено в общий helper.
- Временно включалась CW-диагностика `CW_DEBUG_DISPLAY` на нижней строке LCD.

### Проверено

- `platformio run -e uno`
- RAM: `1422 / 2048 B`
- Flash: `31770 / 32256 B`
