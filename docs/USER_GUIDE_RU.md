## Сборка и прошивка

Основная команда сборки:

```bash
platformio run -e uno
```

Если PlatformIO не может писать в глобальный кеш:

```bash
PLATFORMIO_CORE_DIR=.pio-core PLATFORMIO_SETTING_ENABLE_TELEMETRY=No platformio run -e uno
```

После прошивки старые значения EEPROM могут перекрывать новые дефолты. Текущая
версия этой локальной ветки - `7.01w`; при смене версии настройки EEPROM
пересохраняются автоматически.

## Первый запуск на живой плате

Перед первым включением после прошивки:

1. Соберите baseline `platformio run -e uno` и убедитесь, что размер остался в
   рабочей зоне `32056 / 1396` или близко к ней.
2. Проверьте, что выбран правильный hardware profile:
   для текущего baseline это `USDX_HW_RED_CORNERS`.
3. Если у вашей платы другой экран, LPF или схема PTT, не правьте код внизу
   файла: переключайте только верхний transformer config layer.
4. После прошивки сделайте сброс EEPROM/настроек, если плата раньше работала с
   другой веткой прошивки.

Порядок проверки на столе:

1. Питание без антенны и без внешнего PA.
2. Проверить стартовый экран, валкодер, кнопки и подсветку.
3. Проверить, что частота крутится в правильную сторону; если нет, смотреть
   `SWAP_ROTARY`/hardware preset, а не править обработчик энкодера.
4. Проверить CAT по `115200`, если нужен ПК.
5. Только потом проверять RX, затем TX на минимальном drive и через нагрузку.

Если что-то ведет себя не так:

- неправильная частота или уход по диапазону: сначала проверить `F_XTAL`
  и hardware profile;
- неправильное направление энкодера или диапазонов: сначала проверить
  `SWAP_ROTARY` и `MY_RED_CORNERS`;
- странное поведение подсветки или меню после обновления: сначала сбросить EEPROM;
- проблемы только в нестандартном профиле: пересобрать baseline `uno` и
  сравнить с ним поведение.

## Экран

В обычном режиме верхняя строка показывает частоту в формате `7074.00`, а
справа - компактный S-meter и уровень в dBm, например `S4 -102`.

Нижняя строка в режимах USB/LSB/FM/AM показывает:

- режим (`USB`, `LSB`, `CW`);
- компактные статусы функций;
- полосу фильтра (`FULL`, `3000`, `0500` и т.д.).

Статусы отображаются буквами в фиксированных знакоместах:

| Символ | Значение |
| --- | --- |
| `P/p` | Scan Squelch включен / выключен |
| `R/r` | RIT-смещение активно / неактивно |
| `N/n` | NR включен / выключен |
| `A/a` | AGC включен / выключен |
| `T` | передача |

В CW вторая строка после обозначения режима отдана под декодированный текст.
Если включено редактирование RIT, вместо статусной строки показывается
`RIT +/-xx.xx`.

## Управление

| Действие | Обычный режим |
| --- | --- |
| Вращение валкодера | перестройка частоты выбранным шагом |
| Короткое нажатие валкодера | смена позиции курсора шага частоты вперед |
| Длинное нажатие валкодера | смена позиции курсора шага частоты назад |
| Двойное нажатие валкодера | смена диапазона |
| Удержание валкодера + вращение | громкость, при уходе ниже 0 - power off |
| Короткое нажатие левой кнопки | вход в меню |
| Длинное нажатие левой кнопки | переключение `VFO A/B` |
| Короткое нажатие правой кнопки | смена режима LSB/USB/CW |
| Двойное нажатие правой кнопки | смена полосы фильтра |
| Длинное нажатие правой кнопки | вход/выход из редактирования RIT |

Кнопка валкодера из обычного режима меню не открывает. Это сделано специально,
чтобы не мешать выбору разряда частоты.

## Меню

Верхняя строка меню показывает главный раздел, нижняя - параметр и значение.
Цифровые номера пунктов скрыты.

Разделы:

| Раздел | Что внутри |
| --- | --- |
| `General` | `VOX` (если включен), `QUAD` |
| `RX` | режим, фильтр, AGC, NR, ATT, S-meter, Scan Squelch, SWR, опорная частота, IQ |
| `TX` | practice, drive, PA min/max |
| `CW` | CW decoder, скорость и режим ключа, swap, tone volume |

Навигация:

1. Левая кнопка - открыть меню на последнем выбранном разделе и пункте.
2. Валкодер - выбрать параметр. При прокрутке через край раздела меню
   переходит в следующий или предыдущий раздел.
3. Кнопка валкодера - войти в редактирование выбранного параметра.
4. Валкодер - изменить значение.
5. Кнопка валкодера - сохранить и вернуться к списку параметров.
6. Левая кнопка - выйти из меню; если значение редактируется, сохранить его и
   вернуться к списку параметров.

Во время меню PTT/keyer блокируются, чтобы случайное срабатывание не включало
передачу при настройке.

## Автопрокрутка и остановка по сигналу

Если быстро и равномерно вращать валкодер несколько шагов подряд, прошивка
продолжит перестройку с похожей скоростью. Это инерционная автопрокрутка
частоты.

Любое действие валкодером или кнопкой во время автопрокрутки останавливает ее.

Пункт `RX -> Squelch` управляет автоматической остановкой по уровню сигнала.
В меню показывается не абсолютный уровень, а адаптивный порог относительно
выученного минимума шума: `OFF` или, например, `+6dB`. Скан останавливается,
когда текущий уровень становится выше минимального уровня за этот проход на
заданное число dB.

| Значение на экране | Поведение |
| --- | --- |
| `OFF` | остановка по сигналу выключена |
| `+3dB` | самый чувствительный порог |
| `+7dB` | средний порог |
| `+11dB` | самый строгий порог |

Внутри используется уровень S-meter в dBm. После каждого шага скан ждет короткое
окно обновления уровня, чтобы не остановиться на старом пике от предыдущей
частоты. За текущий проход запоминается минимальный уровень фона, а остановка
происходит на заметном подъеме над минимумом. Если скан останавливается слишком
часто, увеличьте `Scan Squelch`. Если пропускает станции, уменьшите значение.

Во время автопрокрутки экранный `dBm` обновляется чаще, чем в обычном режиме.
Это нужно только для индикации; решение об остановке все равно принимается после
короткой задержки стабилизации на новой частоте. Если уровень похож на станцию,
скан сначала делает короткое подтверждение. Если всплеск пропал, скан идет
дальше. Если сигнал держится, частота удерживается около 10 секунд; если за это
время пользователь ничего не нажал, автопрокрутка продолжается дальше.

Курсор шага частоты во время автопрокрутки скрывается, чтобы LCD не мигал в
нижней строке.

## CW

CW-декодер включен по умолчанию и рассчитан на живой эфир, где паузы между
элементами и буквами не всегда идеальны.

Пункт `CW -> Keyer` выбирает режим:

| Значение | Значение |
| --- | --- |
| `IambicA` | электронный ключ, режим A |
| `IambicB` | электронный ключ, режим B |
| `Straight` | прямой ключ |

`CW Tone` сейчас в меню не показывается, потому что `FILTER_700HZ` отключен в
конфигурации для экономии памяти. Код пункта есть, но включается отдельно.

## CAT

CAT в текущем baseline собран и всегда активен; скорость `115200`.

CAT - самый дорогой по памяти необязательный модуль. Если потребуется большой
запас flash, отключение CAT дает больше всего места, но убирает управление с ПК.
